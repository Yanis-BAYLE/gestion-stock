Nom du projet : StockFlow ou StockManager

Objectif :
Créer une application web de gestion de stock permettant de gérer des produits, fournisseurs, catégories et mouvements d’entrée/sortie,
tout en garantissant l’intégrité des données par des règles métier, des contraintes SQL et des triggers.

Stack :
HTML / CSS / JavaScript
Spring Boot
Spring Data JPA
Spring Security
MySQL 8
Flyway
Docker Compose
JUnit
